# ToDo-Share
This is a simple ToDo List service that aims to handle list sharing.

This project uses the NodeJS framework.

## Install dependencies:
### Middleware: Express
```
npm install express 
npm install @types/express
```
### Database: Prisma (To be Implemented)

### FrontEnd: Angular (To be Implemented)

Change the 'PORT' Variable to the port you want to host
